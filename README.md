# 购物星球导航

一个模仿 EPIC Games Store 风格的购物网站导航页面，专注于展示和导航到各大电商平台。

## 功能特点

- 现代化的深色主题界面
- 左侧固定的分类导航
- 鼠标悬停显示二级分类
- 响应式设计，支持移动端
- 实时搜索功能

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)

## 本地运行

1. 克隆仓库
```bash
git clone [repository-url]
```

2. 打开 index.html 文件
3. 开始使用
